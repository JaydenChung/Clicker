## Purpose

The Security & Privacy Subcommittee advises on protecting sensitive data, managing credentials, and ensuring privacy in the automation system.

---

## Scope

### In Scope
- Credential management
- Data protection
- Privacy considerations
- Threat modeling
- Secure storage
- Session security
- Personal data handling

### Out of Scope
- Detection evasion (→ Anti-Detection)
- Ethical concerns (→ Ethics)
- Browser security specifics (→ Browser Stealth)

---

## Key Questions We Address

1. How should credentials be stored and managed?
2. What sensitive data do we handle and how?
3. What threats should we consider?
4. How do we protect user privacy?
5. What security practices should we follow?
