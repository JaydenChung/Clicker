# ATS Comparison Matrix

## Overview

| ATS | Complexity | Account Required | Typical Pages | Resume Handling |
|-----|------------|------------------|---------------|-----------------|
| Workday | High | Yes | 4-7 | Parse + Profile |
| Greenhouse | Low | No | 1-2 | Upload only |
| Lever | Low | No | 1 | Upload only |
| Taleo | High | Yes | 5-10 | Parse + Manual |
| iCIMS | Medium | Sometimes | 2-4 | Upload + Parse |
| Ashby | Low | No | 1 | Upload only |
| SmartRecruiters | Medium | Optional | 2-3 | Upload + Parse |
| Avature | Medium | Sometimes | 2-4 | Variable |

---

## Workday

### Identification
- URL: `*.myworkdayjobs.com`, `*.wd5.myworkdaysite.com`
- Visual: Blue header, "Workday" branding

### Flow
1. Account creation/login
2. Personal information
3. Work experience
4. Education
5. Skills/certifications
6. Additional questions
7. Review and submit

### Challenges
- Account required
- Multi-page wizard
- Resume parsing often inaccurate
- Many required fields

### Strategy
- Create account first
- Let it parse resume, then correct
- Save profile for reuse

---

## Greenhouse

### Identification
- URL: `boards.greenhouse.io/*`, `job-boards.greenhouse.io/*`
- Visual: Clean, minimal design

### Flow
1. Single page form (usually)
2. Basic info + resume upload
3. Custom questions
4. Submit

### Challenges
- Custom questions vary widely
- Some have many questions

### Strategy
- Fill top to bottom
- Upload resume early
- Answer questions from profile

---

## Lever

### Identification
- URL: `jobs.lever.co/*`
- Visual: Clean, single page

### Flow
1. Single page
2. Resume upload
3. Basic info
4. Submit

### Challenges
- Usually straightforward
- Some have additional links questions

### Strategy
- Very simple, fill and submit

---

## Ashby

### Identification
- URL: `jobs.ashbyhq.com/*`
- Visual: Modern, clean

### Flow
1. Single page form
2. Resume upload
3. Custom questions
4. EEO questions (optional)
5. Submit

### Challenges
- Resume upload usually required
- Some have "Why do you want to work here?" questions

### Strategy
- Upload resume
- Fill profile-based answers
- Handle custom questions

---

## Strategy by ATS Type

### Simple ATS (Greenhouse, Lever, Ashby)
1. Navigate to application
2. Upload resume immediately
3. Fill form fields
4. Handle custom questions
5. Submit

### Complex ATS (Workday, Taleo)
1. Check for existing account
2. Create account if needed
3. Navigate multi-page wizard
4. Let resume parse
5. Correct parsed data
6. Complete all pages
7. Review and submit

### Account-Required Strategy
- Consider pre-creating accounts
- Or handle account creation as soft blocker
- Store credentials securely
